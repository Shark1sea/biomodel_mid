# q1_stats统计说明

**统计结论与解读**  

- **平均发放率: 15.34 Hz（标准差≈1.07 Hz）**: 群体处于稳定的中等放电水平；跨神经元的变异较小，说明网络与外部驱动共同形塑出较为一致的稳态活动。
- **ISI CV: 平均≈0.354（中位数≈0.352）**: 远低于典型泊松过程的 1，表明后神经元发放更为规则，存在时间结构或折返抑制（如不应期、膜动力学反馈）使间隔更均匀。
- **Fano 因子: 平均≈0.206（中位数≈0.167）**: 显著小于 1，计数波动低于泊松噪声，体现出低变异性与一定程度的去噪（可能来自网络整流、突触积分或群体平均化效应）。
- **群体一致性**: 率分布窄、CV 与 Fano 低，说明多数神经元在相似条件下表现出类似的节律与稳定性；无明显双峰或长尾，缺乏强烈的子群分化迹象。
- **机制启示**: 规则化与低噪声通常来源于膜方程非线性、有效不应期、突触时间常数的平滑作用，以及网络耦合对输入噪声的平均化；这些因素共同将随机输入转化为更稳定的输出节律。

**可推导的结论**  

- **非泊松化输出**: 虽然可能受噪声驱动，后神经元群体输出并非泊松，而是更规则的点过程（CV<1、Fano<1）。
- **稳态与可靠性**: 群体在该参数下实现稳态放电，且计数与间隔的变异性均较低，指示较高的输出可靠性与时间可预测性。
- **网络与单体动力学的整流作用**: 神经元内在动力学与网络连接共同降低输入噪声对输出的影响，体现“去噪”与“节律化”功能。
- **实验与模型吻合预期**: 若目标是得到稳定、可控的群体放电（如感知-决策通路的稳态基线或任务期固定频率），当前结果符合此类功能需求。

**后续建议**  

- **参数扫描**: 改变外部驱动强度、突触权重与时间常数，观察 CV/Fano 随条件变化的相图，以定位规则化的机制边界。
- **时间窗敏感性**: 在不同计数窗长度下重算 Fano，以区分短时与长时尺度的变异来源。
- **子群/连接结构**: 引入不同子群或稀疏/抑制耦合，检测是否出现双峰率分布或变异性上升，从而评估结构对稳态与可靠性的影响。
