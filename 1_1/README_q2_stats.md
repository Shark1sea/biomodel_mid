# q2_stats 统计说明与配图

本文说明 `q2_stats.py` 的统计指标含义，并配合生成的图片进行解释。脚本运行后会在同目录生成四张图片与控制台摘要。

## 模型与设置

- 神经元模型：与 `q2.py` 相同的 Izhikevich-风格二变量模型（`v`, `u`），电导型 AMPA 外部与噪声输入。
- 规模与驱动：1000 个同构神经元；外部泊松 100 Hz；独立噪声泊松 500 Hz；模拟时长 1000 ms；`dt=0.05 ms`。

## 输出图片与意义

- `1_1/q2_stats_raster.png`：群体发放栅格图
  - 展示 1000 个神经元在 0–1000 ms 的发放时刻分布。
  - 若图像均匀无明显条纹，表示“异步不规则”（非强同步/非簇发）。

- `1_1/q2_stats_rate_hist.png`：个体发放率直方图
  - 横轴为每个神经元的平均发放率（Hz），纵轴为神经元数。
  - 标题给出群体均值与标准差，例如“均值≈19.15 Hz，标准差≈1.91 Hz”。
  - 体现整体激活水平与个体差异度；同构与整合效应使分布较集中。

- `1_1/q2_stats_isi_cv_hist.png`：ISI 变异系数（CV）分布
  - CV = ISI 标准差 / ISI 均值。Poisson 过程 CV≈1；CV<1 更规则，CV>1 更不规则。
  - 若分布峰值在 0.5 左右，说明膜方程与电导突触时间常数使发放比 Poisson 更稳定（欠分散）。

- `1_1/q2_stats_fano_hist.png`：Fano 因子分布
  - Fano = 计数方差 / 计数均值（在固定时间窗内）。Poisson 过程 Fano≈1。
  - 若大量样本在 0.2–0.4，表示计数波动低于 Poisson，系统呈稳定欠分散，反映不应期与突触整合的平滑作用。

## 控制台摘要

脚本会打印：

- 群体平均发放率与标准差
- ISI CV 的均值/中位数（有效样本）
- Fano 因子的均值/中位数（有效样本）

这些指标共同刻画：

- 在给定噪声驱动下，神经元群体的稳态响应强度与一致性（发放率分布）。
- 时间规律性与不规则程度（ISI CV）。
- 计数层面的分散性（Fano 因子）。
- 当前结果通常表现为“更规则且欠分散”，即比纯 Poisson 更稳定，符合电导型突触与膜动力学的整合作用。

## 运行方式

Windows PowerShell：

```powershell
uv run .\1_1\q2_stats.py
```

生成的图片将保存至 `1_1/` 目录。
